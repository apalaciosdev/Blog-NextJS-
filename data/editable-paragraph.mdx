# Editable paragraph


In this post we learn to create and implement an editable paragraph in our code using **react-addons-update**.

In these examples we are going to change only the state that cointains an array with objects inside.
After this, we can realize a "post" or a "put" http methods to save or update this state in a Database. 


Lorem
We have 3 ways to implement it.

## 1.- Using a text area input
First, we need to add the data of json file in 


```jsx
...
import React, { useState } from 'react'
import update from 'react-addons-update';
...

const [state, setState] = useState(demoData)


const handleChange = (evt) => {
  const { value } = evt.target;

  setState(update(state,{
    [0]:{
      [2]: {
        content:  {$set: value}
      }
    }
  }))
};

return(  
  <div> 
    <textarea type="text" value={state[0][2].content}  onChange={handleChange}></textarea>   
    <p>Renderized State: <span>{state[0][2].content}</span></p>
  </div>
)
```

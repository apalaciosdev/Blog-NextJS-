# Editable paragraph

In this post we learn to create and implement an editable paragraph in our code using **react-addons-update**.

In these examples we are going to change a state that cointains an array with objects inside.

We have 3 ways to implement it.




## 1.- Using an input 

<CodeVisualizer language="js" code="
const contentEditable = createRef()
